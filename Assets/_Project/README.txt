No vamos a hacer carpetas extra, para evitar problemas con archivos repetidos como la vez pasada, por ello
implementaremos la siguiente nomenclatura para cualquier documento que hagamos dentro del proyecto:

Nomenclatura:

Alexco:
	empezar cualquier documento con:_alx_, ejemplo: _alx_Script.cs

Tbox:
	empezar cualquier documento con:_tbx_, ejemplo: _tbx_Script.cs

Ingrid:
	empezar cualquier documento con:_klz_, ejemplo: _klz_Script.cs

Charlo:
	empezar cualquier documento con:_chr_, ejemplo: _chr_Script.cs

Portly:
	empezar cualquier documento con:_ply_, ejemplo: _ply_Script.cs
